# Library Management System (MySQL)


 Files
| File | Description |
|------|-------------|
| **01_create_database.sql** | Creates the `LibraryDB` database and all tables. |
| **02_insert_sample_data.sql** | Inserts sample records into the tables. |
| **03_queries.sql** | Example queries to retrieve and filter data. |
| **04_triggers.sql** | Database triggers for automatic actions. |
| **05_views.sql** | Views for simplified data access. |
| **06_stored_procedures.sql** | Stored procedures for common operations. |

---

 How to Run
1. Open MySQL Workbench and connect to your server.
2. Run the scripts in this order:
   - `01_create_database.sql`
   - `02_insert_sample_data.sql`
   - `03_queries.sql`
   - `04_triggers.sql`
   - `05_views.sql`
   - `06_stored_procedures.sql`
